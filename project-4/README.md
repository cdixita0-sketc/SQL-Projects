# Project 4 - E-Commerce Management System

##  Description

This project is an advanced SQL-based e-commerce system.
It includes multiple related tables such as customers, products, orders, payments, and shipping.

---

##  Tables Used

* Categories
* Products
* Customers
* Orders
* Order_items
* Payments
* Shipping

---
 Operations Performed

* Created database and tables
* Inserted sample data
* Performed complex SQL queries

---

##  Queries Included

* Top 5 highest-priced products
* Total revenue calculation
* Average order value
* Customer order count
* Products not in stock
* Most purchased product

---

##  Advanced Concepts

* Joins (INNER, LEFT, RIGHT)
* Subqueries
* Date functions
* String functions
* Window functions
* CASE expressions

##  Learning Outcome

* Improved SQL query writing
* Better understanding of relationships
* Learned real-world data analysis
